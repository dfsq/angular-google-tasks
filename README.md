# Google Tasks

Google Tasks application built with AngularJs.

## Getting started

1. Clone repository.

2. Install development dependecies:

        npm install
    
3. Run development server:

        grunt connect
        
4. Open address in browser [http://localhost:9999/app][2].

## Demo

You can take a look at this [hosted demo][1].
